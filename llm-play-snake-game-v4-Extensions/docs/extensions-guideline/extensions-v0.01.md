## ğŸ“š **Key Design Patterns**

- **Template Method**: Base classes define structure, extensions implement specifics
- **Factory Pattern**: `GAME_LOGIC_CLS` for pluggable components
- **Strategy Pattern**: Different algorithms/models with consistent interface
- **Extension Over Modification**: Never modify base classes, only extend
